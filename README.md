# MarketPlacePro

## Présentation

MarketPlacePro est une application web de type **marketplace multi‑vendeurs** permettant à plusieurs commerçants de proposer leurs produits sur une plateforme unique. Les clients peuvent rechercher, comparer, commander et payer des articles provenant de différents vendeurs via une interface sécurisée et fluide.

## Objectifs

* Centraliser plusieurs boutiques sur une seule plateforme
* Offrir aux vendeurs un espace de gestion autonome
* Permettre aux clients une expérience d’achat simple et fiable
* Gérer les paiements, commandes et commissions de manière sécurisée

## Fonctionnalités principales

### Utilisateurs

* Inscription et connexion (Client / Vendeur / Admin)
* Gestion du profil utilisateur

### Espace Vendeur

* Tableau de bord des ventes
* Gestion des produits (CRUD)
* Suivi et mise à jour des commandes
* Page publique de la boutique

### Espace Client

* Recherche et filtrage des produits
* Panier multi‑vendeurs
* Paiement sécurisé
* Historique des commandes

### Administration

* Gestion des utilisateurs
* Validation et suspension des vendeurs
* Gestion des commissions

## Technologies utilisées

* **Frontend** : React, Tailwind CSS
* **Backend** : Node.js, Express.js
* **Base de données** : MongoDB ou PostgreSQL
* **Paiement** : Stripe / Paystack / Flutterwave
* **Versioning** : Git, GitHub
